TIPO B

VBA Puro → Google Apps Script

El archivo tiene macros VBA sin dependencias externas (no ODBC, no rutas UNC a servidores). Se convierte usando el motor automático vba_to_gas.py y se revisa manualmente.

Tiempo estimado: 4–8 horas
Complejidad: MEDIA
Archivos BCI: ~2 archivos
Generado: 2026-06-26

Ejemplo Completado — POC001

✅ FORMATEO pp028 (Manuel Díaz · Planes Personas) — POC001 completado 2026-06-20.
Sheet ID: 10c_yP95kH96IACKMjjamcdJyGyyvGZyFIX0ilyumn14

Pasos de Migración

  1. 1

    Scan forense del archivo

    En VAS Home → lupa en el archivo → verificar tipo=B, revisar módulos VBA listados, confirmar que no hay rutas UNC ni ODBC detectados.

  2. 2

    Deconstruir el Excel

    Ejecutar el deconstructor para extraer los módulos VBA y los datos:

    python src/engine/deconstructor.py <ruta_xlsm>

    Resultado en workspaces/OUT_PACKAGES/deconstructed_<nombre>/

  3. 3

    Convertir VBA → GAS automático

    python src/engine/vba_to_gas.py workspaces/OUT_PACKAGES/deconstructed_<nombre>

    Genera 04_SHEETS_PRODUCCION/VBA_Converter/*.gs y un conversion_report.json con el % convertido.

  4. 4

    Revisar y limpiar el GAS generado

    Abrir el .gs generado y revisar: residuos VBA sin convertir, variables const redeclaradas, apóstrofes sueltos. Crear versión limpia en DEPLOY_CLASP/<NOMBRE>.gs.

  5. 5

    Agregar helpers estándar

    Incluir: onOpen() con menú BCI, initConfig(), _logAction(), verLog(). El archivo POC001 FORMATEO.gs es la referencia.

  6. 6

    Deploy via clasp

    cd workspaces/TEST_REAL_01/<carpeta>/DEPLOY_CLASP
npx @google/clasp create --type sheets --title "<NOMBRE> BCI - Visual Sheets v1.0"
npx @google/clasp push --force
  7. 7

    Validar en Google Sheet

    Abrir el Sheet → ingresar datos de prueba → Menú ⚡ BCI Migración → ejecutar macro → verificar hoja Log_<NOMBRE> generada correctamente.

Checklist QA

  • Scan muestra tipo=B, sin rutas UNC, sin ODBC
  • Conversión automática cubre >70% de las líneas
  • Ninguna variable const redeclarada en el .gs
  • Menú ⚡ BCI Migración aparece al abrir el Sheet
  • Macro ejecuta sin errores en Apps Script
  • Hoja de log se crea automáticamente
  • Resultados coinciden con el Excel original (QA comparativo)
  • Sheet ID y Script ID registrados en HITOS.md
