📘 Guías de Migración BCI

Rutas estándar para migrar archivos Excel a Google Workspace. Cada tipo define complejidad, tiempo y pasos.

TIPO A

Sin VBA — Importación Directa

El archivo no tiene macros. Se importa directamente a Google Sheets sin conversión de código. Ruta más simple y rápida.

⏱ 1–2 horas Ver guía →
TIPO B

VBA Puro → Google Apps Script

El archivo tiene macros VBA sin dependencias externas. Se convierte VBA → GAS usando el motor de conversión automático + revisión manual.

⏱ 4–8 horas Ver guía →
TIPO E

VBA + ODBC Sybase PROMETEO_II

VBA con conexión a base de datos Sybase. Requiere bridge REST antes de poder migrar. La mayoría del área COMEX (89 de 97 archivos).

⏱ 2–5 días Ver guía →
TIPO F

Full Pipeline — Máxima complejidad

VBA + ODBC + UserForms + múltiples hojas complejas. El caso más complejo. COMEX tiene 36 archivos de este tipo.

⏱ 3–7 días Ver guía →
